# SaaS Revenue Performance Dashboard (Tableau)

An interactive Tableau dashboard project built to analyze SaaS revenue performance across locations, products, and time.

## Project Overview

This project focuses on revenue analysis using Tableau and includes interactive visualizations to explore:

- Total Revenue by location
- Total Revenue by location and product combination
- Monthly Total Revenue by product
- Monthly ARPPU (Average Revenue Per Paid User)
- Monthly Paid Users Count

The dashboard also includes interactive filters for:

- Location
- Payment Month

These filters are applied across all worksheets to support dynamic analysis and improve dashboard usability.

## Dashboard Features

- **Location-based revenue comparison**
- **Product-level revenue breakdown**
- **Monthly trend analysis**
- **ARPPU performance tracking**
- **Paid user count monitoring**
- **Interactive filters across all visualizations**
